Contains Duplicates

Problem: Given an integer array, return true if any value appears atleast twice in the array

Example:
- Input = [1,2,3,4,4]
- Output = true

Approach:
- Iterate over array, check if element exists in object;
- NO; insert in object
- YES; return true
- Return false if no duplicate found
